# BatSignal

![image](https://user-images.githubusercontent.com/24563791/227809623-6728d92b-ad8c-4087-adf3-42390892a5c7.png)

## Run portable exe
- Check out the latest release at https://github.com/aarontully/BatSignal/releases
- Find the latest version
- Download and unzip the file
- Run the .exe

## How to use
- Select the folder to be monitored in the first input
- If there is a destination folder in the secxond input. Any files created at or moved to the monitored location will then get moved to the second location and deleted from the original source
